## ⚠️ Consulta obrigatória antes de cada sessão

O projeto High Boy é open source. **Antes de qualquer geração de copy, o Agente Leitor deve consultar o repositório GitHub para capturar atualizações recentes** — commits, releases, issues fechadas, PRs mergeados. Essas informações são fonte primária de fatos técnicos reais para os posts.

- Repositório principal: [github.com/HighCodeh](https://github.com/HighCodeh)
- O que buscar: commits recentes, tags de versão, issues fechadas, PRs mergeados, mudanças em README
- Por que: o repo reflete o estado atual do build. Arquivos em `./input/` podem estar desatualizados.
